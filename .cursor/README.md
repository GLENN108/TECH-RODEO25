# Cursor Local Context Directory

This directory contains local context files for the Tech Rodeo project. These files help maintain project context and understanding without relying on cloud storage.

## Files in this Directory

### `project-structure.json`
Complete project structure with file organization, key features, and technologies used.

### `context-summary.md`
Detailed summary of core components, business rules, data flow, and important configuration details.

### `codebase-index.json`
Index of entry points, core modules, dependencies, API integrations, and validation rules.

### `development-notes.md`
Recent changes, known issues, future enhancements, testing checklist, and configuration notes.

### `rules.md`
Project rules, guidelines, code style, business rules, file organization, and security notes.

### `chat-context.json`
Summary of key conversations, important decisions, and project configuration.

## Purpose

These files serve as:
- Project documentation
- Context for AI assistants
- Onboarding reference
- Development guidelines
- Configuration reference

## Note

This is a local context directory. All context is stored locally in this project and does not depend on cloud storage.
